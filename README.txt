Application Artvision sarl

Comment ouvrir l'application :
1. Decompresse le fichier ZIP.
2. Ouvre le fichier index.html dans Google Chrome, Edge ou Firefox.

Comptes de test :
- admin / admin123
- papeterie / papier123
- aluminium / alu123
- imprimerie / print123
- electronique / elec123
- light / light123

Important :
Cette version fonctionne sur un seul ordinateur ou dans un navigateur.
Pour plusieurs utilisateurs en ligne avec une vraie securite, il faudra connecter
l'application a une base de donnees en ligne comme Supabase.
